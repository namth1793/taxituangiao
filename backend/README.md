# Backend – Web Taxi Tuần Giáo

Website này là landing page tĩnh (HTML + CSS thuần), **không cần backend**.

## Cách chạy

Chỉ cần mở file `../frontend/index.html` trực tiếp trên trình duyệt.

## Nếu muốn deploy lên server

Chép file `frontend/index.html` lên bất kỳ hosting tĩnh nào:
- GitHub Pages
- Netlify (kéo thả file)
- VPS: đặt vào thư mục `/var/www/html/` của Nginx/Apache

## SEO Local – Gợi ý thêm

- Đăng ký Google Business Profile với từ khóa "taxi tuần giáo"
- Thêm schema markup `LocalBusiness` vào thẻ `<head>` nếu cần
- Tạo sitemap.xml sau khi có tên miền
